# Dlkitgo Examples

This folder contains practical examples for using dlkitgo, a free library for accessing popular platforms without cookies, authentication, or payment.

## Available Examples

Currently, the collection includes examples for the following platforms:

*   **Spotify:** [examples/spotify](/examples/spotify)
*   **Instagram:** [examples/instagram](/examples/instagram)

To use an example, go into the specific platform's folder and follow the instructions in its `README.md` file.

## Provider Request

If you would like to see an example for a platform that is not listed here, you can request it by opening an issue.

Make a new [issue](https://github.com/Beesonn/dlkitgo/issues/new?assignees=&labels=new+provider+request&template=request.yml) with the name of the provider on the title, as well as a link to the provider in the body paragraph.
